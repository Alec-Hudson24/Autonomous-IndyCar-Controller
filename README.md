# Autonomous-IndyCar-Controller

-  The goal of this project was to design a controller for an Indy Car, allowing it to autonomously go around a track using GPS way points, the types of controllers used on this project were,
    -  Proportional
    -  Proportional Integral
    -  Proportional Integral and Derivative

 -  This project required the use of:
    -  Control Laws
    -  Matlab
    -  Matlab plugins (such as Simulink)
